Fedora 删除多余内核

经常用dnf更新的话，两三周grub2就会有一堆启动项。一个大概就100MiB，多余内核在grub2里面看着也十分碍事，下面是终端命令删除多余内核。

grub2登陆时记住当前使用的内核版本号，或者之后用uname查看也可以。
$sudo rpm -aq|grep kernel
打出内核的详细列表，列表内容只能手动一个一个删除。
例如
$sudo dnf remove -y kernel-core-4.9.10………………
也可以用正则表达式批量删除。
