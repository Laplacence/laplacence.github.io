C++中的异步操作

实际上，C++11中对异步的支持不比Python和JavaScript弱，就是有点长。

一个小插曲，这里编译的时候博主用g++直接-o，结果疯狂出错。最后发现要加上-pthread参数才能正常使用，看来gcc操作也要多学学啊。。。
