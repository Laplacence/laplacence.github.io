Oracle VM VirtuaBox/Mininet安装过程

注意：这篇文章是在Fedora 31的环境下完成的。
mininet现在在github上面的building虽然是passing的状态，但开发组现在推荐的方式和版本是2.2.2以及使用虚拟机并采用他们调试完成的镜像来进行mininet的操作。

github上找到mininet的release，里面就有他们调试好的ubuntu mininet镜像，下载。
首先去VirtualBox官网下载Fedora所需要的rpm安装包，wget或者图形界面都可以。
趁这个时候可以先把内核的开发者sdk下载安装了，使用dnf直接下载即可。

安装完rpm之后启动，把镜像放入，可能会提示启动失败，这时候参照错误提示用root身份进行操作即可。
如果操作完依旧会弹出错误提示（例如会提示你使用root运行/sbin/vboxconfig，依赖的编译器都有的情况下依旧building失败了），建议更新至最新的VirtualBox，即可解决该问题。
